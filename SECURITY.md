# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | :white_check_mark: |
| < 0.3.0 | :x:                |

## Reporting a Vulnerability

Axon Pro is built by **Technicodes**. We take security seriously. If you discover a vulnerability, please report it via email to:

**security@technicodes.com**

Please include:
- A description of the vulnerability.
- Steps to reproduce.
- Any potential impact.

We will acknowledge your report within 24 hours and provide a timeline for resolution.
